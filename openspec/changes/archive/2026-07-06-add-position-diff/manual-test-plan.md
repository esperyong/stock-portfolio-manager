# 调仓 diff（v2）手工测试用例

> 对应 tasks 3.1。逐项执行，通过则勾选。用例中的具体数值来自 2026-07-06 的真实接口数据：
> 2025 年数据已定稿应稳定复现；2026-03-31 期的行数/个股以实际披露为准。
> 测试基金：**163417 兴全合宜混合A**（2025 年有完整 4 期，含港股与北交所标的，覆盖所有场景）。

## 〇、前置准备

- [ ] P1 拉取分支并启动：`git fetch && git checkout feat/add-position-diff && npm install && npm run tauri dev`
- [ ] P2 数据准备：进入「基金跟踪」页 → 添加 **163417**（搜「兴全合宜」选 A 份额）。若此前已添加过，点一次「刷新」即可
- [ ] P3 确认数据就绪：展开卡片 →「调仓」页签 → 期次下拉应有 **5 期**：
  `2026-03-31（前十大）`、`2025-12-31（全量·116只）`、`2025-09-30（前十大·14只）`、`2025-06-30（全量·101只）`、`2025-03-31（前十大·21只）`
  ——只要 2025 年 4 期齐全且全量/前十大标记正确即通过（本项同时验证了**双年拉取**新行为）

## 一、默认对比与基础交互

- [ ] A1 **默认期次**：打开「调仓」页签，两个下拉默认为 `2025-12-31 → 2026-03-31`（最新两期，起始在左）
- [ ] A2 **默认对比的口径语义**（全量→前十大）：
  - 顶部出现蓝色提示条，且**只**提到"目标期为前十大口径：「退出披露」不代表清仓…"（不应提"起始期"那句，因为起始期 12-31 是全量）
  - 明细中消失类条目的 Tag 显示 **退出披露**（不是"清仓"）
  - 出现类条目的 Tag 显示 **新建仓**（起始期是全量名单，此前确实未持有——这是"按方向判定"的关键点）
- [ ] A3 **持平项开关**：默认不显示持平项；若存在持平项，表头旁有「显示持平项（N）」勾选框，勾选后表格追加持平行
- [ ] A4 **起止相同**：把两个下拉选成同一期 → 弹出警告「起止报告期不能相同」，表格不刷新

## 二、全量 vs 全量（真实清仓/新建仓语义）

操作：起始选 `2025-06-30（全量·101只）`，目标选 `2025-12-31（全量·116只）`。

- [ ] B1 **无提示条**：两侧均为全量，蓝色口径提示条不出现
- [ ] B2 **分类数量级**（该基金下半年大换仓，数字应接近）：新建仓 ≈ **87** 只、加仓 ≈ **6**、减仓 ≈ **18**、清仓 ≈ **72**、持平 ≈ **5**
- [ ] B3 **新建仓抽查**：`300308 中际旭创` 为新建仓，且排在新建仓组最前（权重变化约 **+8.1pp**，类内按 |Δ权重| 降序）
- [ ] B4 **加仓抽查**：`300750 宁德时代` 加仓，持股数变化约 **+36%**（权重约 +2.9pp）；`688008 澜起科技` 加仓约 **+194%**
- [ ] B5 **清仓抽查**：`01810 小米集团-W`（约 -6.3pp）、`00981 中芯国际`、`00268 金蝶国际` 均为 **清仓** Tag（目标侧全量，可确证）——同时验证港股前导零代码正常显示
- [ ] B6 **排序**：分组顺序为 新建仓 → 加仓 → 减仓 → 清仓（→ 持平），组内大变动在前
- [ ] B7 **红绿方向**：新建/加仓的百分比为红色系，减仓/清仓为绿色系（A 股习惯红涨绿跌）

## 三、前十大 vs 全量（按方向判定的核心用例）

操作：起始选 `2025-09-30（前十大·14只）`，目标选 `2025-12-31（全量·116只）`。

- [ ] C1 **提示条只提起始侧**："起始期为前十大口径：「新进披露」不一定是新建仓…"（不应出现"目标期"那句）
- [ ] C2 **可确证的清仓**：`00981 中芯国际`（9 月末还在前十大）Tag 显示 **清仓**——目标侧是全量名单，消失即真清仓；`920808 曙光数创`（北交所 6 位代码）同为清仓
- [ ] C3 **不可确证的新进**：大量条目（约 104 只）Tag 显示 **新进披露**（不是"新建仓"——起始侧只披露了前十大，无法确证此前未持有）
- [ ] C4 **股数口径的价值**（顺带观察）：`09969 诺诚健华` 应为 **加仓**（股数约 +6%）但其权重变化为**负**（约 -0.7pp）——权重口径会误判为减仓，这正是股数主口径的意义

## 四、边界与降级

- [ ] D1 **断网只读**：断开网络 → 切换任意期次组合 → 对比正常刷新（纯读库）；点「刷新」按钮 → 中文报错，已有数据与对比功能不受影响；恢复网络
- [ ] D2 **单期引导文案**（可选）：添加一只 2026 年新成立的基金（搜索新发基金，成立不足半年的）→ 若其只有一期数据，「调仓」页签应显示"该基金目前只有一期持仓数据…"引导文案而非报错。若找不到合适标的可跳过（双年拉取后老基金都有多期）
- [ ] D3 **股数缺失回退**（条件触发）：若某对比行出现「按权重估算」小 Tag，确认其股数列显示"—"且加减仓判定与权重方向一致。2025 年数据通常股数齐全，此项无出现则视为 N/A
