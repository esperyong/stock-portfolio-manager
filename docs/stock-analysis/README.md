# 个股分析报告库 · Stock Analysis

本目录汇总各只个股的价值投资深度分析报告。每只个股一个子文件夹，内含 Markdown 与 HTML 两份同名报告。

## 目录与命名规范

```
docs/stock-analysis/
  └─ <本名>-<代码>/           # 一只个股一个文件夹
      ├─ <本名>-<代码>.md      # Markdown 报告（正文）
      └─ <本名>-<代码>.html    # 自包含 HTML 可视化报告（明暗主题自适应，可直接浏览器打开）
  README.md                    # 本索引
```

**命名按市场用其本名（语言随市场切换）：**

| 市场 | 语言 | 文件夹示例 |
|---|---|---|
| A 股（沪深） | 中文 | `特变电工-600089` |
| 港股 | 中文（本名） | `泡泡玛特-09992` |
| 美股 | 英文 | `Apple-AAPL` |

- 文件夹与 md/html 文件**同名**；报告内以 `> 研究时间：YYYY-MM-DD` 标注日期（日期不进文件名，便于原地更新）。
- HTML 为手工构建的自包含单文件（内联 CSS/SVG，无外部依赖），风格暖色纸感 + 明暗主题。
