裁决：回避 / 持有不追高

好生意，但现价在为"多晶硅复苏"定价——没有安全边际

核心的输变电特许权 + 低成本煤是优质资产，"十五五"电网 4 万亿投资（+40%）+ 出口放量给了多年可见度。但在 ~21.5 元：现价 ≈ 或略高于中性内在价值（~16–18 元），无安全边际；赔率不对称偏不利（上 +35% vs 下 −49%）；到 2028 年报告自由现金流仍是负到勉强转正（三年累计仅 ~+20 亿）。

巴菲特/芒格式买点在中位数十几元（约 12–15 元）——届时净拿输变电+煤炭盈利能力，把多晶硅周期当免费期权。用别人恐惧多晶硅的时候，去买它的输变电和煤。

01它是做什么的

特变电工由张新在新疆创办，本质是围绕新疆廉价能源 + 电力工程能力搭起来的四块业务。2025 年营收 972.3 亿元（基本持平），归母净利润 59.5 亿元（+43.7%）。

⚠️ 盈利质量提示：+43.7% 有水分——扣非净利润仅 45.5 亿（+15.6%），差额约 15.8 亿是华电新能上市的一次性公允价值收益；2026Q1 扣非净利甚至同比 下滑约 3.8%。头部增速高估了真实经营改善。

02盈利的本质：营收不动，利润随商品价大起大落

营收四年几乎不动（960–980 亿），净利润却在 159 亿(2022)→41 亿(2024)→60 亿(2025) 间剧烈摆动——波动全部来自多晶硅与煤炭价格。而自由现金流已连续两年为负：因为资本开支不降反升到 221 亿，公司把赚的钱几乎全部再投资。

03上下游产业链

🔌 输变电（利润引擎）

上游：取向硅钢（外购，宝钢/首钢主导，属成本风险）、铜、铝（期货套保）。下游：国网/南网、特高压、出口。换流变压器国内第一（~34%）。增量：沙特 164 亿大单、国际签约 20 亿美元(+68%)、海外订单 >50 亿美元。

☀️ 多晶硅/新能源

上游：工业硅 + 电力（占成本 41.5%，故自备电厂是关键）。下游：硅片/光伏，纯商品、严重过剩。2025 产量 −51.5%、利用率仅 37%。特变现金成本 ~3 万/吨，有竞争力但非行业最低。

⛏️ 煤炭 + 发电（一体化根基）

准东煤田储量 126 亿吨、产能 7,400 万吨（申请核增至 1.64 亿吨）。开采成本仅 66–100 元/吨 vs 全国 150–200，自有 5,040MW 煤电。自有廉价煤 → 便宜电 → 低成本硅/铝，是护城河来源。

🧪 新材料（铝）

能源 → 氧化铝 → 高纯铝 → 电子铝箔 → 电极箔（下游铝电解电容器）。稳定但利润占比小。

04未来 1–3 年自由现金流预测（2026–2028）

报告口径自由现金流（经营现金流 − 资本开支，亿元）。资本开支仍在高位是 FCF 迟迟难以转正的主因。

年份悲观中性乐观核心驱动
2026−90−35+25输变电强+沙特放量；多晶硅约盈亏平衡；资本开支仍 190 亿高位
2027−55+5+90沙特 164 亿执行、出口占比 ~30%；资本开支降至 180 亿，FCF 首次转正
2028−15+50+135建设期基本结束、资本开支回落 155 亿，FCF 才真正有意义
三年累计（中性）—≈+20—即便基准兑现，股东可见自由现金依然微薄
关键结论：公司持续每年砸 100–130 亿增长性资本开支（相当部分投向"无护城河"的多晶硅扩产）。你买的不是打折的自由现金流，而是一个仍在重资本投入期的半商品综合体。我的归母净利中性预测 60/68/76 亿，低于券商共识 74/90/107–125 亿（我按多晶硅盈亏平衡建模，共识把周期反转当基准）。

05内在价值与安全边际

股东盈余 DCF（股权成本 11–12%，归一化盈余锚 48–52 亿）与分部估值(SOTP)交叉验证。经对抗式核验，原模型两处偏差（折现率偏低、归一化偏高）已下修，均把内在价值推向更低。

现价对应静态 PE 约 18 倍、扣非 PE 约 24 倍。~21.5 元还是从 52 周低点 11.4 元反弹上来的——几个月前存在的安全边际已被这波上涨吃掉。

06巴菲特—芒格框架评估

护城河 混合

🟢 宽护城河：输变电是真特许权（换流变压器国内第一、全产业链、绑定两网、出口领先）——电网建设与全球电气化的"收费站"。
🟡 成本资产：准东低成本煤，好资产但周期性。
🔴 无护城河：多晶硅纯商品、严重过剩，成本仅二线，一体化可复制——下行期的价值毁灭者。

管理层 尚可

🟢 创始人控制、长期导向；建成输变电特许权与低成本煤；主动减产控产显示纪律。
🔴 逆周期扩张多晶硅 +26 万吨（投过剩）；连续两年 FCF 深度为负、负债升至 499 亿；利润被一次性收益美化。分红温和波动（股息率 ~1.7%）。非"回报导向"画像。

能力圈 太难

单看输变电/煤矿可理解；但合并盈利被极难预测的多晶硅价格主导（300→34 元/公斤，净利 159→41→60 亿）。叠加四板块、港股子公司、大额少数股东、一次性损益——正常化盈余极难锚定，芒格大概率"放进太难堆"。

价格 无边际

现价 21.5 元 ≥ 中性内在价值 16–18 元。不满足"以合理价买优秀公司"。合理买点在中位数十几元（12–15 元，约 30% 折让）。

07主要风险

  • 多晶硅价格再度探底：2026 年中 N 型料 ~33 元/公斤已跌回现金成本附近，若"反内卷"供改落空，新特重回亏损甚至减值（2024 曾计提 34 亿）。
  • 自由现金流长期为负 + 负债上升：FCF −43(2024)/−127(2025)；有息负债 499 亿且在增。悲观情形三年累计"烧钱" ~160 亿，有真实资产负债表压力。
  • 盈利质量：2025 高增长靠一次性收益，2026Q1 扣非已负增长。
  • 综合体复杂性 / 大额少数股东：264.6 亿少数股东权益、港股子公司、频繁重述，正常化盈余难估。
  • 政策与执行：电网投资政策性、有波动；沙特订单仅 ~70% 锁定、需本地建厂，有执行/汇率/地缘风险。
  • 煤炭周期 + 能源转型：煤炭毛利已从 32% 降至 22%，逆风侵蚀成本优势。

08数据与置信度说明

  • FY2025 核心财务（营收 972.3 / 归母 59.5 / 经营现金流 93.3 / 资本开支 221 / FCF ≈ −127 亿、股本 5.05 亿股、市值 ~1,080 亿）经独立对抗式核验确认，无"亿/万"单位混淆。
  • 一处已修正：夹带资料曾把新特持股写成 76%，正确为 64.43%（模型已用正确值，不影响估值）。
  • 较低置信度：2026-07-23 收盘价 21.53 元及当日涨幅无法用一手行情源独立确认（7 月成交价验证在 20 出头区间，故合理）；新特港股市值未纳入 SOTP（用账面估算）。
  • 估值模型经核验下修两处（折现率改用股权成本、归一化盈余下调），两者均把内在价值推低，强化而非削弱"回避"结论。
