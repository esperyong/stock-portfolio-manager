# 儿子日本券商开户与 NISA 投资规划（讨论纪要）

> 讨论时间：2026-07-05。背景：儿子 21 岁（成年），东海大学札幌校区留学生。
> 目标：开一个可交易美股+日股的券商账户并绑定银行卡，建立长期投资配置。
> 本文为完整讨论结论，供后续执行与复查。

## 一、最终方案（TL;DR）

| 项目 | 结论 |
|---|---|
| 券商 | **楽天証券**（开户时一并开通 **新 NISA** + 特定口座·源泉徴収あり） |
| 主银行 | **みずほ銀行**（孩子自选；与楽天証券有官方「らくらく入金」免费即时入金通道，瑞穗持有楽天証券约 49% 股份） |
| 备用银行卡 | **ゆうちょ銀行**（北海道 Seicomart 店内即ゆうちょATM，全道存取款方便，补瑞穗在道内网点薄的短板） |
| 核心仓（70%~90%） | 不分配型 S&P500 指数基金（eMAXIS Slim 米国株式 S&P500 或 楽天・S&P500），つみたて投資枠定投 + 楽天卡积分返现 |
| 卫星仓（10%~30%） | 美元高股息现金流：**VYM**（真美元分红）或 **楽天SCHD·受取型**（增配成长），二选一或六四开 |
| 明确排除 | 日股高息 ETF（1489 等）——日元贬值敞口是首要顾虑，尽管其在 NISA 内分红零税漏 |

**两条关键实操铁律**：
1. 分配型基金在 NISA 内**必须选「受取型」**——选"再投資型"会额外消耗当年 NISA 额度
2. 分红设置选「**株式数比例配分方式**」——分红进券商账户才享受 NISA 免税

## 二、开户前置条件（外国籍留学生）

- **在留卡**：在留资格「留学」，申请时有效期须**剩余 2 个月以上**（各券商通用要求；快到期先续签再开户）
- **My Number（个人番号）**：登记住民票后自动分配，开证券账户必需
- 外国籍开证券账户目前**只能邮寄开户**，全程约 1~2 周
- 网银（楽天銀行等）要求**入境满 6 个月**（外汇法"居住者"认定）；瑞穗只要求在留期剩余 3 个月以上，另需出示**学生证或在籍证明**
- 若刚入境不满半年：先开ゆうちょ（在留 3 个月以上即可）→ 满 6 个月后再开网银/券商

## 三、券商与银行选择的依据

### 费用（2026 年时点，SBI 与楽天已基本拉平）

| 费用项 | 楽天証券 | SBI証券 |
|---|---|---|
| 日股佣金 | 0 | 0 |
| 美股佣金（课税账户） | 0.495%，上限 $22（超割コース返 1% 积分） | 同左 |
| **美股佣金（NISA 内）** | **0** | **0** |
| 日元⇄美元实时换汇 | **0 钱** | **0 钱** |

费用已无差异，选择回归生态：瑞穗×楽天証券的官方「らくらく入金」通道（2024 年起，免费即时，仅楽天银行与瑞穗两家享有）是本次选楽天的决定性因素。

### 瑞穗银行的已知短板（接受）

- 北海道网点极薄（基本只有札幌市中心一家支店），便利店 ATM 取现收 110~330 日元/次 → 用ゆうちょ卡补
- 无楽天银行マネーブリッジ式的自动扫款联动 → 入金手动点一下，可接受
- 附属借记卡是 JCB，海外通用性弱；纸质存折 1,100 日元/年（选电子存折免费）
- 瑞穗侧维护窗口：周六 22 点～周日 8 点不能入金

## 四、新 NISA 要点（2024 年起永久化）

日本个人投资免税制度：正常资本利得+分红税 20.315%，NISA 内 **0%，免税期无限**。

| | つみたて投資枠 | 成長投資枠 |
|---|---|---|
| 年度额度 | 120 万日元 | 240 万日元 |
| 可买 | 金融厅审核的低成本指数基金（约300只，定投方式） | 日股、**美股个股/ETF**、大部分公募基金 |

- 终身免税上限 **1,800 万日元**（成长框最多 1,200 万），按买入成本计
- **卖出后额度次年恢复**（按成本价恢复）
- 每人同一时间只能在**一家**机构开 NISA（每年可换一次）→ 券商要选定再开
- 2026 年度税制改正：新增 0~17 岁こどもNISA、定投框品种扩容（与本案无直接关系，说明制度持续加码）

### 留学生四个关键点

1. **资格**：18 岁以上 + 日本居住者（住民票+My Number）即可，外国籍留学生可开
2. **美股分红在 NISA 内漏 10%**：美国按美日税收协定源头预扣 10%；因日本端税率为 0，无法用外国税額控除抵回（课税账户则可以抵）。**到手 = 分红 × 90%，分红本体完全可自由支取**——拿不回的只是税，不是分红
3. **損益通算不可**：NISA 内亏损不能与课税账户盈利互抵 → 高风险仓放课税账户，稳健长期仓放 NISA
4. **毕业离日**：变为非居住者后 NISA 不能再买入，网络券商一般要求销户；在住期间已实现的免税收益不受影响 → 离境前规划好卖出/清算

学生附加好处：NISA 内收益不计入合计所得，不影响被扶养认定、国保保费与留学生收入申报。

## 五、S&P500 配置（核心仓）

- **つみたて枠**：eMAXIS Slim 米国株式（S&P500）（约 0.08%/年）或 楽天・S&P500（约 0.077%/年），100 日元起投，分红基金内部自动再投资（不分配型）
- **楽天カード定投**：每月上限 10 万日元，按卡种返 0.5%~1% 楽天积分
- **成長枠**可现货申购同款基金（一次性加仓用），也可买 VOO/IVV/SPY 或东证上市 S&P500 ETF（2558/1655，日元计价年 2 次分红）
- 不分配型 vs 分红型总回报几乎等价（内部同样被美国预扣 10% 股息税），差别只在剩余 90% 自动滚存还是发现金；**攒钱目标下不分配型复利效率最高**
- S&P500 股息率本身仅 ~1.1%~1.5%/年（美股 ETF 为季度分红），主要回报来自股价上涨；追求现金流不该用 S&P500，应配高股息产品

## 六、高股息卫星仓（现金流）

### 美股四大高息 ETF（2026 年初约值）

| ETF | 股息率 | 费率 | 特点 |
|---|---|---|---|
| VYM | ~2.4% | 0.06% | ~580 只超分散，最稳，涨价能力最强 |
| HDV | ~3.2% | 0.08% | ~75 只质量筛选，防御型 |
| SPYD | ~4.4% | 0.07% | 股息最高但波动大、行业集中、有减配史——**不推荐学生持有** |
| SCHD | ~3.9% | 0.06% | 连续分红10年+的约100家，**增配之王**（年增配 ~10%） |

- **SCHD 原版在日本买不到**，通过 **楽天SCHD**（楽天・シュワブ・高配当株式・米国ファンド 四半期決算型）持有：费率 0.1238%，每年 2/5/8/11 月分配，NISA 成長枠可买，**务必选受取型**
- 日股高息 ETF（如 1489，股息率 ~3.5-4%、年 4 回、NISA 内分红零税漏）在税务上最优，但**因日元贬值敞口被排除**——家庭财富以非日元计价，2020 年至今日元对美元贬值约三成，4% 股息率不敌汇率损耗

### SCHD vs VYM 选择逻辑

- 第一优先级是"真美元现金流"（分红直接以美元落账、零操作滚存美元）→ **VYM**
- 第一优先级是"20 年后每年收多少分红"（yield on cost 随增配复利，持有 10 年后对成本股息率可达 8-10%）→ **SCHD**
- 楽天SCHD 的日元分红**不构成经济劣势**：底层就是美元资产，到账后在楽天証券内 0 钱换汇手动转回美元即可（一年 4 次、每次一键），仅是便利性差异
- 边界认知：楽天証券账户内的美元只能账内使用（再投资/换回日元出金），**不能以美元提现到银行**；若未来要"美元离开券商"，那是 SBI+住信SBI 组合的强项

## 七、楽天积分（ポイント）说明

- 1 积分 = 1 日元的楽天生态代金券；卡定投返现来源
- **不能提现、不能直接换美元**
- 最优用法：**ポイント投資**——在楽天証券内用积分直接买 S&P500 基金/楽天SCHD，积分即刻转化为美元资产
- 量级预期：月定投 3 万日元 × 普卡 0.5% ≈ 年 1,800 积分，是甜头不是收益来源
- 楽天卡留学生可申请（18+），信用白纸可能被拒；拒卡不影响证券账户，改用银行扣款定投即可（无积分）

- 实际分工：开户后的一次性设置（NISA、定投、分红方式）在 Web 完成；日常看行情/交易股票 ETF 用 iSPEED；看基金持仓用 iGrow

### 各地域可用性（重要）

| 地点 | 结论 |
|---|---|
| 日本 | 全功能正常，唯一被官方支持的使用地 |
| 新加坡 / 美国（短期旅行） | 网络可达；官方口径"不保证支持"，**居住者临时使用实务上被容忍**（见下），登录大概率触发风控认证 |
| 中国大陆 | **按"不可用"规划**：网络可达性不稳定 + 中国区商店无 App + 认证短信收取困难，三重障碍 |

- **官方口径**：楽天証券仅在日本持牌，[海外使用可能抵触当地法规](https://www.rakuten-sec.co.jp/web/support/procedures/non-resident/)，服务原则上面向日本国内；长期出国须办「出国手续」（转非居住者，NISA 停止买入）
- **技术现实**：从境外登录会触发[リスクベース認証](https://www.rakuten-sec.co.jp/web/security/risk_based_authentication/)（风控加验），需要日本手机号接收验证——**出国前保持日本 SIM 可国际漫游收码**，否则可能被锁在门外
- **化解之道——自动定投不需要登录**：定投+卡扣款设置好后每月自动执行，人在任何国家都不影响投资运转。所以正确姿势是：调仓、卖出、设置变更等主动操作安排在日本境内完成，短期离日期间让账户"自动驾驶"
- **安全设置**：2025 年日本各券商发生大规模账户盗用事件后风控全面收紧——开户后务必开启全部追加认证（登录追加认证、App 生物识别），并且只从官方商店装 App、不点短信里的链接（钓鱼是盗用主要入口）

### 境外登录操作的合规性分层（含 VPN 问题）

短期回国/出境旅行不改变日本居住者身份（税务与 NISA 资格均不受影响），问题出在"从境外主动下单"这个动作上，分三层：

**1. 券商合约层（读准条款的措辞层级）**

券商的红线画在「**居住者 vs 非居住者**」，而不是「人在境内 vs 境外」：
- SBI 对海外访问的官方口径是"以国内使用为前提、可能无法使用/部分服务可能临时停止"——这是"**不保证支持**"，不是禁止；"原则不可交易"的硬性表述只针对**成为非居住者**的客户。楽天"可能抵触出国先法令"警告的主语同样是非居住者
- **居住者临时出差/休假期间登录交易：不违法，条款上属"自担风险"地带，实务普遍且被容忍**——日本出差族/度假客大量存在此场景，券商为此建有海外风控解锁流程与 App 设备记忆机制（用装好的 App 登录比在当地新浏览器登录少触发风控），大额操作可出发前向客服报备
- 明确越线的两个动作：**① 用 VPN 伪装成日本 IP**（刻意绕过风控 = 违反利用规约；券商能识别数据中心 IP 与设备指纹，2025 年盗号潮后极敏感，后果是账户冻结/限制，人在国外时解锁流程非常痛苦）；**② 非居住者隐瞒身份继续交易**（"以旅行之名行长居之实"——连续数月从同一海外国家登录交易的模式）

**2. 所在地法律层（两地截然不同）**
- **新加坡**：个人用 VPN 合法；游客操作自己的海外券商账户，当地法律不管（MAS 监管在新展业的机构，不管过境个人）。当地法律层面基本干净，只剩券商条款问题
- **中国大陆**：多一个独立问题——**个人翻墙本身即违反《计算机信息网络国际联网管理暂行规定》**（实践中以行政处罚为主、执行零星，但确属违规而非灰色默许）。整个操作链条变成：违规动作（翻墙）→ 叠加违反券商条款（境外下单），没有一环站得住。作为日本居住者虽不触及中国跨境证券监管红线（那些规定针对券商向大陆居民展业），但不改变前述结论

**3. 实务结论**

| 场景 | 建议 |
|---|---|
| 新加坡短期停留，查看行情/持仓 | 没问题，直连即可（多半触发一次风控加验） |
| 新加坡短期停留，偶发下单 | 当地法律风险≈无，居住者身份下实务被容忍；姿势：**直连 + 手机 App + 日本 SIM 漫游收码，不要用 VPN**（伪装 IP 反而是加重情节），大额行前报备客服 |
| 中国大陆，任何主动操作 | **放弃**。VPN 违规 + 条款违规 + 收码困难三重叠加，不值得 |
| 通用正解 | 行前把操作做完；定投自动执行无需登录；真紧急（必须卖出）时接受风控代价直连操作一次，事后配合券商核验 |

**关键认知**："用 VPN 保证网络稳定"与"用 VPN 伪装成日本"是两回事，但券商风控无法区分动机——它只看到 IP 与真实位置不符。所以即使在新加坡，**直连（真实 IP + 接受风控加验）才是比 VPN 更"合规"的姿势**。

### 家人代操作的红线（与"本人旅行中交易"性质完全不同）

设想场景：父母在海外登录账户代为交易，儿子（名义人）在日本转发验证码——**不可行**，踩三条实线：

1. **本人以外利用禁止**：日本券商约款明令禁止名义人以外使用账户（家人也不例外），证券业协会规则禁止**借名取引**。且该模式风控画像极明显（海外 IP 持续主动交易 + 名义人手机始终在日本收码 = "操作者非本人"），后果是冻结/强制销户级别；转发验证码等于定向击穿 KYC 机制
2. **税务实质归属**：出资与决策都来自父母时构成**名義株/名義預金**——资产实质归属父母，儿子的 NISA 免税资格被非本人资产"挪用"（NISA 严格限定名义人本人投资），赠与/遗产层面归属混乱；名義預金是日本国税厅遗产税稽查的常规重点
3. **中国侧**：中国税务居民通过亲属境外账户实际操作证券投资 = 绕开个人境外证券投资管制（合规通道为 QDII/港股通等），亲属代持正是监管的典型打击画像

**合规替代做法**：

| 需求 | 做法 |
|---|---|
| 给儿子账户注入资金 | 正常赠与：日本赠与税基础扣除**每年 110 万日元**内免税免申报（注意：投资本金不属于抚养义务的生活费/学费豁免范围，须计入赠与额度） |
| 参与投资决策 | 建议随便提（"这个月加仓 VYM"），**下单的手必须是儿子本人** |
| 父母自己的钱投美股 | 走自己名义的合规渠道（QDII 标普500 基金、本人合规的港美股账户），不与儿子身份绑定 |

**一句话边界总结**：儿子本人 + 日本境内 = 全绿；儿子本人 + 临时海外直连 = 灰色但被容忍；任何"非本人操作" = 红线，无论人在哪。

> 以上为基于公开监管框架与券商条款的分析，不构成法律意见。未来涉及大额资产或长期跨境安排时，建议咨询懂中日两地的跨境税务/合规律师。
