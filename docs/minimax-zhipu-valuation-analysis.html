10 倍市值差来自五重叙事 / 资金面 / 股东结构溢价——而非收入：

01商业模式溢价

市场给"2B/2G 私有化 + MaaS 经常性收入 + AGI 基座第一股"付高溢价；给 2C 消费应用（充值/token、留存波动大）折价。

02国资光环 + 港股通

智谱一批地方国资背书、被当"国家队"，且已进港股通能吃南向资金；MiniMax 被挡在最大增量买家门外直到 8 月。

03流通盘与筹码

解禁前 MiniMax 流通盘极小易逼空；智谱解禁大头是国资、明确长期持有、抛压小；MiniMax 的 VC/基石有强烈变现诉求。

04叙事 + 情绪

智谱=最接近 OpenAI 的国产标的；MiniMax 近期踩雷——大幅涨价、被 Anthropic 指"蒸馏"、M3 基准被质疑，压制估值。

回应"智谱 coding plan 卖爆 → 2026 营收反超"

方向对，但"远超"高估了差距。智谱 GLM Coding Plan 是最扎实的商业化飞轮——全球付费开发者破 24.2 万、Token 6 个月涨 15 倍，涨价 30–60% 后调用量仍 +400%，GLM-5.2 比 Claude 便宜约 80% 持续售罄，云端收入 +293%；券商预测 2026E 营收 ~28 亿元。但 MiniMax 并不弱：2025 增速 +159%（快于智谱 +132%）、ARR 已超 $1.5 亿、73% 收入来自海外。即便智谱 2026 营收更高，量级约为 MiniMax 的 1.5–2.5 倍，远小于 ~8 倍的市值差——收入解释不了 10 倍鸿沟，溢价仍在倍数与叙事。

支持买的理由（多头）：

01相对估值

P/S ~150x vs 智谱 ~950x，若板块继续被追捧或价差收敛，MiniMax 向上空间更大。

028/6 港股通催化剂

03已挤掉大量泡沫

较峰值 −78%（1330→~290），今日解禁抛压尾盘被承接，说明有接盘力量。

04产品有真东西

海螺视频、Talkie 出海、3 亿+ 用户、100 万+ 企业客户、B 端收入占比 >30%。

需警惕（空头）

① 仍比发行价贵约 80%——追进去是买在反弹上；② 解禁抛压未尽：46% 流通盘刚放出，VC/基石有变现动机，后续还有波次；③ 基本面仍贵：150x P/S + 2025 亏 133 亿且扩大、现金消耗快；④ 叙事劣势：无国资光环、2C 非 2B/2G、信任事件缠身；⑤ "利好兑现即卖出"：入通预期部分已反映，历史上正式入通后常见获利了结。

若要参与，理性做法：

  1. 仓位要小——当高风险博弈仓，不当核心持仓。
  2. 等抛压衰竭——看接下来几天/几周是否放量见底、日线企稳，别在解禁潮里接飞刀。
  3. 8/6 前布局——但别把身家押在单一催化剂上。
  4. 认清代价——你是在为一家仍在扩大亏损的 2C 公司付 ~150 倍市销率，"比泡沫便宜"不等于"便宜"。

现价 偏谨慎 / 中性：这是一笔围绕 8/6 港股通的事件型高风险博弈，不是价值入场。

若"MiniMax vs 智谱二选一"，我反而认为智谱下行风险更大（~950x 极端估值 + 刚解禁 + 回 A 扰动），只是它有国资 / 港股通 / 国家队叙事托底。两者更可能的收敛路径是 智谱下杀 而非 MiniMax 补涨——请把这点放进你的仓位假设里。
