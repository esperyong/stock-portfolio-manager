# MiniMax vs 智谱：解禁暴跌后的估值差与买入评估

> 研究时间：2026-07-09。标的：**MiniMax（稀宇科技，00100.HK / MINIMAX-W）** vs **智谱（02513.HK）**。
> 问题：今日 MiniMax 迎上市后首轮解禁、盘中一度暴跌 20%——现价是否值得买入？同为国产大模型头部公司，为何 MiniMax 市值与智谱差了近 10 倍？MiniMax 的发行价是多少、按发行价市值几何？它是否尚未开通港股通？
> 方法：多路网络检索 + 关键数字交叉核验。价格/市值为 2026-07-09 盘中快照，随行情变动；财务口径（币种、年度、含义）已逐处标注。本文为投研结论，不构成投资建议。

## 二、先回答三个事实问题

### 1. 发行价与发行市值

- **发行价：165 港元/股**，定在招股区间上限（公开发售超额认购 1837 倍、国际发售 37 倍，认购火爆）。
- **按发行价算的市值：约 500 亿港元**（招股书发行估值区间约 461–504 亿港元，定价在上限附近，约合人民币 455 亿）。
- 上市首日（2026-01-09）+109%、收 345 港元、市值突破 1050 亿；2026-03 一度冲到约 1330 港元、市值约 4100 亿。
- 今日盘中低点 289.6 港元、市值约 908 亿——**较峰值回撤约 78%，但较发行价仍高约 75–80%**。追现价者，是买在比 IPO 贵近一倍、比峰值便宜约八成的位置。

### 2. 港股通——判断正确，MiniMax 尚未纳入

原因是 **WVR（同股不同权）架构**，港交所对双重股权公司审查更严，需满足附加门槛：**上市满 6 个月 + 20 个交易日、过去 183 交易日日均市值 ≥ 200 亿港元、期间总成交额 ≥ 60 亿港元**，因此 **预计最早 2026-08-06 才能开通港股通**。而智谱没有 WVR 限制，**已于 2026-06-08 与恒生科技指数同步纳入港股通**。

这是两者估值差的关键因素之一——**这轮 AI 双雄行情主要由南向资金推动，而南向现在根本买不了 MiniMax**。

### 3. 今日解禁的准确画面

今日为 MiniMax 上市后首轮解禁，**解禁 1.07 亿股，流通盘从约 5% 一次性放大到约 46%**（供给近乎放大 10 倍）。IPO 前 14 家基石合计占 67.9%，均设定今日解禁。盘中一度跌超 20% 至 289.6 港元、市值跌破千亿，**但尾盘被买回、收涨约 +3.75%**——说明有承接盘，供给冲击当日被吸收。对照之下，智谱 7/8 解禁当天涨超 13%（解禁大头为国资基石且明确长期持有、抛压小），今日 7/9 再涨约 15%（盘中 +20%），并趁势按 1588 港元配售新 H 股融资。

## 三、为何估值差近 10 倍？——不是收入差出来的

| 维度 | **MiniMax（00100）** | **智谱（02513）** |
|---|---|---|
| 上市日 / 发行价 | 2026-01-09 / 165 港元 | 2026-01-08 / 116.20 港元 |
| 发行市值 | ~500 亿港元 | ~511 亿港元 |
| **当前市值** | **~900–1000 亿港元** | **~7000–8000 亿港元**（峰值曾破万亿） |
| **2025 营收** | **~5.6 亿元**（$79M，**+159%**） | **7.24 亿元**（+132%） |
| **2026E 营收** | ~11–18 亿元（ARR 已超 $1.5 亿 ≈ 11 亿元，推算） | **~28 亿元**（券商预测） |
| **市销率 P/S** | **~150x** | **~800–950x** |
| 商业模式 | **2C 为主**：Talkie / 星野 / 海螺AI（陪伴、视频、语音）+ API | **2B/2G 为主**：MaaS，私有化部署占 73.7%，ARR ~17 亿 |
| 股东底色 | VC / 财务投资人为主（14 家基石占 67.9%） | **重国资**：中关村、杭州、大兴、华发、成都高新、上海国资… |
| 港股通 | 未纳入，预计 8/6 | **已纳入（6/8）** |
| 2025 亏损 | IFRS 亏 **$1.87B**（**主要为优先股公允价值变动等非现金项**，IPO 后消失，≠ 经营现金消耗） | 经调整净亏 31.82 亿元 |

**核心洞察：智谱收入只比 MiniMax 高约 1.3 倍，市值却高约 8 倍。** 也就是说 10 倍的市值差，几乎全部由估值倍数解释，而非基本面收入。这个倍数差来自五重溢价：

1. **商业模式溢价**：市场给"2B/2G 私有化部署 + MaaS 经常性收入 + 国产 AGI 基座第一股"的确定性叙事付高溢价；给 2C 消费应用（充值 / token 变现、留存与付费波动大）折价。
2. **国资光环 + 港股通/南向资金**：智谱背后一批地方国资，被当"国家队"；且已进港股通、能吃南向资金——这轮行情的主力买家。MiniMax 现在南向买不了，被挡在最大增量买家门外，直到 8 月。
3. **流通盘与筹码**：解禁前 MiniMax 流通盘极小、易被逼空；智谱解禁大头是国资基石且明确长期持有、抛压小；MiniMax 的 VC / 基石解禁后有强烈变现诉求（有报道直言"投资人开始想如何让 LP 利益最大化"）。
4. **叙事定位**：智谱 = "全球大模型第一股 / AGI 基座"，被当成最接近 OpenAI 的国产标的；MiniMax = 多模态消费应用（视频 / 语音 / 陪伴），当下市场更爱前者。
5. **情绪受损**：MiniMax 近期踩雷——大幅涨价（用户成本最高涨 2.5 倍，与同行降价反向）、被 Anthropic 指"蒸馏"、M3 基准被质疑，压制估值。

**换句话说：与其说"MiniMax 太便宜"，不如说"智谱贵得离谱"**——智谱 ~800–950x 的市销率远超 OpenAI（约 40x 参照）。MiniMax 只是相对这个泡沫便宜，不等于绝对便宜。

### 补充：智谱 coding plan 卖爆 → 2026 营收会反超吗？

**方向是对的，但"远超"高估了差距。**

**智谱这边（支持该判断）**：GLM Coding Plan 是它最扎实的商业化飞轮——全球付费开发者已破 **24.2 万**、Token 调用量 6 个月涨 **15 倍**；2026-02 GLM-5 发布后结构性涨价 30–60%，涨价后调用量仍 **+400%**；GLM-5.2（2026-06-17 开源旗舰）定价比 Claude 便宜约 80%，作为"Claude Code 平替"持续售罄（算力吃紧、供不应求）。云端部署收入同比 **+293%**，商业模式从项目制转向 token 计费。券商预测 **2026E 营收约 28 亿元**（2027E ~70 亿、2028E ~158 亿）。这也是智谱"业务质量"胜过 MiniMax 的真实理由：经常性、高毛利（2026E 毛利率回升至 ~43%）。

**但 MiniMax 并不弱（对"远超"的修正）**：2025 营收 +158.9%（**快于智谱的 +132%**）；截至 2026-02 **ARR 已超 $1.5 亿**（≈11 亿元）、开放平台 2 月新注册用户为去年 12 月的 4 倍；海螺 AI 2025 前 9 个月营收暴增 644%；**73% 收入来自海外**（地域结构更分散）。

**关键结论**：即便智谱 2026 营收更高，量级大概率是 MiniMax 的 **~1.5–2.5 倍**，而非市值反映的 ~8 倍。**收入差（≤2.5x）远小于市值差（~8x）——所以营收解释不了 10 倍鸿沟，溢价仍主要在估值倍数、叙事与资金面。** 这个推断强化了、而非削弱了本文的核心判断：智谱贵在"倍数"，不在"收入"。

## 四、是否值得现价买入？——事件驱动的高风险博弈，不是价值入场

### 多头理由

1. **相对估值**：P/S ~150x vs 智谱 ~800–950x，若板块继续被追捧或两者价差收敛，MiniMax 向上空间更大。
2. **8/6 港股通催化剂明确**：一旦纳入，南向资金可买，是这轮行情最强的边际买家。彭博测算潜在南向资金池很大，大摩估两家合计带来约 12.5–17.5 亿美元被动增量。
3. **已从峰值回撤约 78%**（1330 → ~290），泡沫挤掉不少；今日解禁抛压尾盘被承接，说明有接盘力量。
4. **产品面有真东西**：海螺视频、Talkie 出海、3 亿+ 用户、100 万+ 企业客户、B 端收入占比 >30%。

### 空头 / 需警惕

1. **仍比发行价贵约 80%**——"暴跌"是相对峰值，不是相对成本；今日收盘其实是涨的，追进去是买在反弹上。
2. **解禁抛压远未结束**：46% 流通盘刚放出，VC / 基石有变现动机；后续还有解禁波次，motivated sellers 会持续压制。
3. **基本面仍很贵**：150x P/S + 2025 亏 133 亿且扩大、现金消耗快、2C 变现更难。
4. **叙事劣势**：无国资光环、2C 非 2B/2G，且有 Anthropic 蒸馏指控 / 涨价争议等信任事件。
5. **催化剂可能"利好兑现即卖出"**：入通预期部分已反映，历史上"正式入通后获利了结"很常见（券商已提示）。

### 我的判断

**不是"抄底捡便宜"，而是围绕 8/6 港股通的事件型博弈。** 若要参与，理性做法是：① 仓位要小、当高风险博弈仓，不当核心持仓；② 等解禁抛压衰竭再动——看接下来几天/几周是否放量见底、日线企稳，别在解禁潮里接飞刀；③ 在 8/6 前布局但别把身家押在单一催化剂上；④ 清醒认知：你是在为一家仍在扩大亏损的 2C 公司付 ~150 倍市销率，"比泡沫便宜"不等于"便宜"。

**若是"MiniMax vs 智谱二选一"**：我反而认为智谱的下行风险更大（~800–950x 的极端估值 + 刚解禁 + 回 A 预期扰动），只是它有国资 / 港股通 / 国家队叙事托着。两者更可能的收敛路径是**智谱下杀**而非 MiniMax 补涨——这点要放进仓位假设里。
